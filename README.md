# Elden Horses
Minecraft Fabric Mod that adds netherite horse armor. Once equipped, horses obtain abilities similar to those found in Elden Ring. [Curseforge Page]()

## Abilties
* Double Jump
* Hotkey to summon horse
* Horse disappears on dismount
